# Wiki Conflicts

Этот файл фиксирует противоречия между источниками, требующие ручного разрешения.

## Формат записи

```
### [ДАТА] [PROJECT] — [ТЕМА]
- **Источник A**: `raw/<project>/<file>` — <позиция>
- **Источник B**: `raw/<project>/<file>` — <позиция>
- **Статус**: open | resolved
- **Решение**: <описание, если resolved>
```
